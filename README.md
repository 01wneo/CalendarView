# CalenderView
Android上一个优雅、高度自定义、性能高效的日历控件，支持标记、自定义颜色、农历等。Canvas绘制，速度快、占用内存低
##Gradle
```
compile 'com.haibin.calendarview:1.0.0'
```
```
<dependency>
  <groupId>com.haibin</groupId>
  <artifactId>calendarview</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
